Level A
Q1. How would you create a hyperlink (a link) to another website in HTML?
You can create a hyperlink using the <a> tag with the href attribute. For example: 
<a href="https://www.example.com">Visit Example</a>
Q2. Explain the difference between inline elements and block elements in HTML.
Inline elements do not start on a new line and only take up as much width as necessary, whereas block elements start on a new line and take up the full width available. Examples include <span> (inline) and <div> (block).
Q3. Describe the use of the <blockquote> tag in HTML.
The <blockquote> tag is used to denote a section that is quoted from another source. It typically renders with an indentation. 
<blockquote>This is a quote.</blockquote>
Q4. What is the purpose of using the <em> tag in HTML? Provide an example.
The <em> tag is used to emphasize text, usually rendered in italics. 
<p>This is an <em>emphasized</em> text.</p>
Q5. How do you add attributes to HTML elements? Provide an example.
You can add attributes directly in the opening tag of an element. For example: 
<img src="image.png" alt="Description">
Q6. Create one webpage and show the use of the style attribute.
You can use the style attribute to add CSS styles directly to an element. 
<p style="color: blue;">This text is blue.</p>

Level C

Q34. Discuss the purpose of the <meta> tag in the <head> section of an HTML document. Explain how to use it for search engine optimization (SEO) and social media sharing. Provide examples of relevant attributes, such as description and keywords.
The <meta> tag is a fundamental component of the <head> section in an HTML document. It provides metadata about the HTML document itself, which is essential for browsers and search engines. The purpose of the <meta> tag includes specifying the character set, the author's name, and, most importantly for SEO, the description and keywords of the web page. For search engine optimization (SEO), the <meta name="description"> attribute plays a crucial role. It provides a concise summary of the web page's content, which appears in search engine results. A well-crafted description can significantly influence click-through rates. The <meta name="keywords"> attribute, though less impactful today, can still be used to list relevant keywords related to the content. Additionally, when sharing on social media platforms, the <meta> tag can enhance the appearance of links shared. Attributes like og:title, og:description, and og:image are part of the Open Graph protocol, which allows for more engaging content previews. Example of using the <meta> tag for SEO and social media sharing: 
<meta charset="UTF-8">
<meta name="description" content="Learn the basics of HTML and how to build your first web page.">
<meta name="keywords" content="HTML, web development, coding">
<meta property="og:title" content="My First Web Page">
<meta property="og:description" content="An introduction to HTML basics.">
<meta property="og:image" content="image.png">
Q35. What is the role of the <link> tag in an HTML document? How can it be used to link to external stylesheets, and what are the benefits of using separate stylesheets? Provide examples.
The <link> tag serves as a connection point between an HTML document and external resources, particularly stylesheets. By using the <link> tag, developers can include CSS styles that enhance the presentation of a web page without cluttering the HTML structure. Linking to an external stylesheet promotes a clear separation of content (HTML) and presentation (CSS), making the codebase more maintainable. This modular approach allows developers to make style changes without altering the HTML file. Furthermore, using separate stylesheets can significantly improve loading times as browsers can cache CSS files, reducing the need to download the same styles on subsequent visits. Example of linking an external stylesheet using the <link> tag: 
<link rel="stylesheet" href="styles.css">
In this example, the browser will load the styles defined in styles.css to apply to the HTML document. Using external stylesheets enhances collaboration among developers, allows for easier updates, and promotes consistency across multiple pages of a website.
Q36. Provide an example of using the <canvas> element in HTML. Discuss its purpose and how it can be manipulated with JavaScript for dynamic graphics creation.
The <canvas> element is a powerful feature in HTML that allows for dynamic graphics generation via JavaScript. It creates a blank rectangular area on the web page, which can be drawn on using various methods provided by the Canvas API. This element is widely used for creating game graphics, data visualizations, and other graphical content. To utilize the <canvas> element, you must specify its dimensions through the width and height attributes. After defining the canvas, JavaScript is used to access the drawing context, which provides methods for rendering shapes, text, images, and more. Example of using the <canvas> element: 
<canvas id="myCanvas" width="200" height="100"></canvas>

<script>

    var canvas = document.getElementById('myCanvas');

    var ctx = canvas.getContext('2d');

    ctx.fillStyle = "blue";

    ctx.fillRect(0, 0, 200, 100);

</script>
In this example, a blue rectangle is drawn on the canvas. The getContext('2d') method specifies a two-dimensional drawing context, allowing the use of methods like fillRect() to create shapes. The Canvas API also supports advanced operations like animations and user interactions, making it an essential tool for modern web development.
Q37. Create an HTML document structure that includes the following elements: a title with the text "My First Web Page", a heading with the text "Welcome to HTML Basics", and two paragraphs: one containing "HTML stands for HyperText Markup Language" and the other containing "It is used to structure content on the web."
Below is a structured example of a simple HTML document that showcases various fundamental elements. The document includes a title, a heading, and two paragraphs that explain what HTML is and its purpose. The basic structure of an HTML document begins with the <!DOCTYPE> declaration, which informs the browser about the version of HTML being used. The <html> tag wraps the entire document, while the <head> section contains metadata, including the document's title. The <body> section is where the content displayed on the page is defined. Here is the complete code: 
<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>My First Web Page</title>

</head>

<body>

    <h1>Welcome to HTML Basics</h1>

    <p>HTML stands for HyperText Markup Language.</p>

    <p>It is used to structure content on the web.</p>

</body>

</html>
This code effectively introduces the reader to HTML basics by providing essential information in a clear format, demonstrating the hierarchy of content from the title to the paragraphs.
Q38. Add an audio element with controls that plays the audio file "song.mp3". Provide alternative text "Play the song" for the audio element.
The <audio> element in HTML5 is designed to embed sound content in web pages. This element can include various audio file formats, such as MP3, WAV, and OGG, making it versatile for web applications. By providing controls, users can play, pause, and adjust the volume of the audio directly within the web interface. When implementing the <audio> element, it's important to include the controls attribute, which adds a set of audio controls to the interface. Additionally, the <source> element can be used within the <audio> tag to specify the path to the audio file and its type. Below is an example of how to implement an audio element that plays "song.mp3": 
<audio controls>

    <source src="song.mp3" type="audio/mpeg">

    Play the song

    </audio>
In this example, the text "Play the song" serves as fallback content for browsers that do not support the audio element. This ensures that users still have context even if the audio cannot be played, promoting better accessibility and user experience.
Q39. Style the heading from the previous question to have a red color and a font size of 24px. Additionally, italicize the second paragraph. Create an unordered list with three items: "Apples," "Bananas," and "Oranges."
Styling elements in HTML is essential for enhancing the visual appeal of web content. CSS (Cascading Style Sheets) is used to apply styles to HTML elements, allowing developers to control the layout, colors, fonts, and overall presentation. To style the heading with red color and a font size of 24px, and to italicize the second paragraph, we can use CSS selectors. The h1 selector targets the heading, while the p:nth-of-type(2) selector targets the second paragraph. Here’s an example of how to apply these styles, along with an unordered list containing three items: 
<style>

    h1 { color: red; font-size: 24px; }

    p:nth-of-type(2) { font-style: italic; }

    </style>

    <ul>

        <li>Apples</li>

        <li>Bananas</li>

        <li>Oranges</li>

    </ul>
This code applies the specified styles and generates a list of fruit items. The use of an unordered list (<ul>) creates a bulleted list, enhancing the readability and organization of the content.
Q40. Create an HTML form that allows users to submit their name and email. The form should have the following elements: a text input for the user's name, an email input for the user's email address, and a submit button with the label "Submit Form." Apply the following attributes: set the form method to "POST," and add a label for each input field using the <label> element.
Forms are essential in web applications as they enable user interaction, allowing visitors to input and submit data. An HTML form can consist of various input types, including text, email, and buttons. By organizing form elements with <label> elements, we enhance accessibility and usability, providing clear descriptions for each input field. Below is an example of an HTML form that collects a user's name and email address. The form is set to use the POST method for data submission: 
<form method="POST">

    <label for="name">Name:</label>

    <input type="text" id="name" name="name"><br>

    <label for="email">Email:</label>

    <input type="email" id="email" name="email"><br>

    <input type="submit" value="Submit Form">

</form>
In this example, the form consists of labeled inputs for the user's name and email, as well as a submit button. The for attribute in the <label> elements links each label to its corresponding input field, improving accessibility for screen readers and enhancing the overall user experience. Upon submission, the form data will be sent to the server for processing.
