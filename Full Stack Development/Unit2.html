Q1. In how many ways can CSS be integrated into a web page (types of CSS)?
CSS can be integrated into a web page in three main ways:
  1. Inline CSS: This involves using the style attribute within HTML tags. It applies styles directly to an element, but it's not recommended for maintaining a clean codebase.
  2. Internal CSS: Styles are defined within a <style> tag in the <head> section of the HTML document. This method is useful for styling a single document.
  3. External CSS: An external stylesheet is linked to the HTML document using the <link> tag. This approach is preferred for larger websites as it allows for easier maintenance and consistency across multiple pages.
Q2. What is the syntax to link the external style sheet?
To link an external stylesheet to an HTML document, you use the following syntax within the <head> section: 
<link rel="stylesheet" href="styles.css">
In this example, styles.css is the path to the external CSS file.
Q3. What is a hyperlink? Explain internal and external hyperlinks.
A hyperlink, often referred to as a link, is a clickable reference in a web page that directs users to another resource. There are two main types of hyperlinks:
  • Internal Hyperlink: This links to another page within the same website. For example: <a href="about.html">About Us</a>
  • External Hyperlink: This links to a page on a different website. For example: <a href="https://www.example.com">Visit Example</a>
Q4. List 5 features of JavaScript.
Here are five key features of JavaScript:
  • Dynamic Typing: Variables can hold values of any data type, and types can change at runtime.
  • Object-Oriented: JavaScript supports objects and inheritance, allowing for modular code.
  • Event-Driven: JavaScript responds to user actions like clicks and key presses, making web pages interactive.
  • Cross-Platform: JavaScript runs on any device with a web browser, regardless of the operating system.
  • Rich Interfaces: JavaScript can create animations, interactive forms, and dynamic content to enhance user experience.
Q5. Explain the difference between client-side and server-side JavaScript.
Client-Side JavaScript: This code is executed in the user's web browser. It is primarily used for creating interactive user interfaces, handling events, and manipulating the DOM. Examples include form validation and dynamically updating content without refreshing the page.

Server-Side JavaScript: This code runs on the web server and is used for handling requests from the client, interacting with databases, and performing server-side logic. Node.js is a popular environment for running server-side JavaScript. It allows for the development of scalable network applications.
Q6. What are different types of variables in JavaScript?
JavaScript supports three types of variables:
  • var: This declares a variable that can be re-declared and updated. It has function scope and is hoisted.
  • let: This declares a block-scoped variable that can be updated but not re-declared within the same scope. It is preferred for use in modern JavaScript.
  • const: This declares a block-scoped variable whose value cannot be changed after assignment. It must be initialized during declaration.
Q7. What is the function of a document.write() method?
The document.write() method in JavaScript is used to write content directly to the HTML document. It is primarily used for debugging and testing purposes. However, it should be avoided in modern web development as it can overwrite the entire document if called after the page has loaded. 
document.write("Hello World!");
Q8. Explain prompt(), confirm(), alert() functions in JavaScript.
These are built-in JavaScript functions for user interaction:
  • alert(): Displays a simple alert dialog with a message. It has an OK button.
  • confirm(): Displays a dialog with a message and two buttons: OK and Cancel. It returns true if OK is clicked, and false if Cancel is clicked.
  • prompt(): Displays a dialog with a message prompting the user for input. It returns the input value or null if Cancel is clicked.
Q9. What are the different variations of CSS?
CSS has several variations, including:
  • CSS1: The first version of CSS, which introduced basic styling features.
  • CSS2: Enhanced features like positioning, media types, and improved selectors.
  • CSS3: The latest version, which adds support for new layout models (Flexbox and Grid), animations, transitions, and responsive design techniques.
Q10. What is a CSS selector? Write names of 5 CSS selectors.
A CSS selector is a pattern used to select the elements you want to style. Here are five types of CSS selectors:
  • Type Selector: Selects all elements of a specified type (e.g., div, p).
  • Class Selector: Selects elements with a specific class attribute (e.g., .className).
  • ID Selector: Selects an element with a specific id attribute (e.g., #idName).
  • Attribute Selector: Selects elements with a specified attribute (e.g., [type="text"]).
  • Descendant Selector: Selects elements that are descendants of a specified element (e.g., div p selects all p elements inside div elements).
Q11. What is the use of CSS Opacity?
The CSS opacity property is used to specify the transparency level of an element. It accepts values from 0.0 (completely transparent) to 1.0 (completely opaque). For example: 
opacity: 0.5;
This would make the element 50% transparent.
Q12. Explain universal selector.
The universal selector in CSS is represented by the asterisk symbol (*) and selects all elements on the web page. It is often used to apply a style universally or to reset default styles. For example: 
* { margin: 0; padding: 0; }
This code resets the margin and padding for all elements to zero.
Q13. Name the property for controlling the image repetition of the background with code.
The property used to control the repetition of background images in CSS is background-repeat. Here’s an example: 
background-repeat: no-repeat;
This prevents the background image from repeating.
Q14. What is the difference between inline, embedded, and external style sheets?
The differences are as follows:
  • Inline Style Sheets: Styles are applied directly within HTML elements using the style attribute. Example: <h1 style="color: red;">Title</h1>
  • Embedded Style Sheets: Styles are defined within a <style> tag in the <head> section of the HTML document. This affects only the current document.
  • External Style Sheets: Styles are written in separate CSS files and linked to the HTML document using the <link> tag. This allows for consistent styling across multiple pages.
Q15. What is CSS Box model and what are its elements?
The CSS Box Model describes the rectangular boxes generated for elements in the document tree and consists of the following elements:
  • Content: The actual content of the element, where text and images appear.
  • Padding: The space between the content and the border, transparent by default.
  • Border: The area that surrounds the padding (if any) and content.
  • Margin: The outermost layer that creates space between the element and surrounding elements.
Q16. What is the float property of CSS?
The float property in CSS is used to position elements to the left or right of their containing element, allowing other content to flow around them. For example: 
float: left;
This would float the element to the left side of its container.
Q17. What is the purpose of the z-index and how is it used?
The z-index property in CSS controls the vertical stacking order of positioned elements (elements with a position value of relative, absolute, or fixed). A higher z-index value means the element will appear above those with lower values. For example: 
z-index: 10;
This sets the stacking order of the element to 10.
Q18. Explain the difference between visibility: hidden and display: none?
The differences are:
  • visibility: hidden: The element is hidden but still takes up space in the layout. It is not visible to the user but remains part of the document flow.
  • display: none: The element is completely removed from the document flow, meaning it takes up no space in the layout. Other elements will occupy the space where it was previously rendered.

Q20. Explain Min height and Max height property in CSS.
The min-height property in CSS sets the minimum height of an element. It ensures that the element does not shrink below a specified height, regardless of its content. For example: 
min-height: 100px;
The max-height property sets the maximum height of an element. It prevents the element from exceeding a specified height. For example: 
max-height: 200px;
Q21. What does the ‘a’ in rgba mean?
In rgba, the 'a' stands for "alpha," which represents the opacity level of the color. The rgba() function takes four parameters: red, green, blue, and alpha (opacity), where alpha ranges from 0 (completely transparent) to 1 (completely opaque). For example: 
rgba(255, 0, 0, 0.5)
This represents a semi-transparent red color.
Q22. What are CSS HSL Colors?
HSL stands for Hue, Saturation, and Lightness. It is a way to represent colors in CSS:
  • Hue: Represents the color type on the color wheel, ranging from 0 to 360 degrees.
  • Saturation: Represents the intensity of the color, with 0% being a shade of gray and 100% being the full color.
  • Lightness: Represents the brightness of the color, with 0% being black and 100% being white.
The syntax for HSL colors is hsl(hue, saturation, lightness).
Q23. What are the different CSS border properties?
The main CSS border properties include:
  • border-width: Specifies the width of the border.
  • border-style: Specifies the style of the border (e.g., solid, dashed, dotted).
  • border-color: Specifies the color of the border.
  • border-radius: Rounds the corners of the border.
You can also use the shorthand property border: to set all three properties at once. For example: 
border: 2px solid red;
Q24. What does margin: 40px 100px 120px 80px signify?
The margin: 40px 100px 120px 80px; shorthand notation specifies the margin values for all four sides of an element in the following order:
  • Top margin: 40px
  • Right margin: 100px
  • Bottom margin: 120px
  • Left margin: 80px
Q25. What is the difference between margin and padding?
The differences between margin and padding are:
  • Margin: The space outside an element, separating it from other elements. It is transparent and does not affect the element's background.
  • Padding: The space inside an element, between the content and the border. It is part of the element's background and affects its overall size.
Q26. What are the different CSS link states?
The different CSS link states include:
  • link: The default state of a link.
  • visited: The state of a link that has been visited by the user.
  • hover: The state when the user hovers over the link.
  • active: The state when the link is being clicked.
These states can be styled individually using CSS.
Q27. What are the various positioning properties in CSS?
The main positioning properties in CSS are:
  • static: The default positioning; elements are positioned according to the normal document flow.
  • relative: Positions an element relative to its normal position.
  • absolute: Positions an element relative to its nearest positioned ancestor (not static).
  • fixed: Positions an element relative to the viewport; it remains in the same position even when scrolling.
  • sticky: A hybrid of relative and fixed positioning; the element toggles between relative and fixed depending on the scroll position.
Q28. What does the CSS float property do?
The float property in CSS is used to allow an element to float to the left or right of its container, enabling text and inline elements to wrap around it. It can be used for layout purposes. For example: 
float: right;
This will float the element to the right side of its container.
Q29. What does display: inline-block do?
The display: inline-block; property in CSS allows an element to be treated as an inline element while maintaining its block properties. This means the element can have width and height, and it will sit next to other inline or inline-block elements. It also allows for margins and paddings to be applied without affecting the surrounding elements' layout.
Q30. How can we make our website responsive using CSS?
To make a website responsive using CSS, you can:
  • Use relative units such as percentages or `em`/`rem` instead of fixed units like pixels.
  • Utilize media queries to apply different styles based on screen size or device type. Example: 
    • @media (max-width: 600px) { ... }
  • Implement a flexible grid layout with CSS Grid or Flexbox to adapt to different screen sizes.
  • Set images and other media to have a max-width of 100% to ensure they scale down appropriately.
Q31. What is CSS flexbox?
CSS Flexbox (Flexible Box Layout) is a layout model that allows you to design a complex layout structure in a more efficient and predictable way than traditional models. It provides the ability to align items, distribute space, and rearrange elements within a container, regardless of their size. It is particularly useful for responsive design, as it enables elements to grow and shrink to fit the available space.

Q33. What is the difference Between Programming, Scripting, and Markup Languages?
The differences are as follows:
  • Programming Languages: Used to create software applications. They are typically compiled and executed to perform complex tasks. Examples include C++, Java, and Python.
  • Scripting Languages: A subset of programming languages that are interpreted rather than compiled. They are often used for automating tasks or adding interactivity to web pages. Examples include JavaScript, PHP, and Ruby.
  • Markup Languages: Not programming languages; they are used to define the structure and presentation of text in documents. Examples include HTML (for web pages) and XML (for data interchange).

Q34. What are CSS backgrounds? List the properties with Syntax?
CSS backgrounds are used to set the background for an element. The main properties for CSS backgrounds are:
  • background-color: Sets the background color of an element.
  • background-image: Specifies an image as the background.
  • background-repeat: Determines how the background image is repeated.
  • background-position: Sets the starting position of a background image.

Syntax:

element {
    background-color: color;
    background-image: url('image.jpg');

    background-position: position;
    background-size: size;
}
Q35. How can we format text in CSS? Write with Syntax?
Text can be formatted in CSS using various properties to control its appearance:
  • font-family: Specifies the font of the text.
  • font-size: Defines the size of the font.
  • font-weight: Sets the thickness of the font.
  • text-align: Aligns the text (left, right, center, justify).
  • line-height: Sets the space between lines of text.
  • color: Defines the text color.

Syntax:

element {
    font-family: 'Font Name';
    font-size: size;

    text-align: left | right | center | justify;
    line-height: number | normal;
    color: color;
}
Q36. Write a JavaScript Program in an HTML document to Print the Fibonacci Sequence.
Here is a simple program to print the Fibonacci sequence up to a specified number: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fibonacci Sequence</title>
</head>
<body>
    <script>
        function fibonacci(n) {
            let a = 0, b = 1, temp;
            document.write("Fibonacci Sequence: ");
            for (let i = 0; i < n; i++) {
                document.write(a + " ");
                temp = a;
                a = b;
                b = temp + b;
            }
        }
        fibonacci(10); // Print first 10 Fibonacci numbers
    </script>
</body>
</html>

Q44. What is the main difference between var, const, and let? Explain With program.

Answer: The main differences between var, let, and const in JavaScript are:

  • Scope: var is function-scoped or globally scoped, while let and const are block-scoped.
  • Hoisting: Variables declared with var are hoisted to the top of their scope and initialized with undefined, while let and const are hoisted but not initialized.
  • Reassignment: Variables declared with let can be reassigned, while const cannot be reassigned after its initial assignment.

Example:


var x = 10;
if (true) {
    var x = 20; // Same variable
    console.log(x); // 20
}
console.log(x); // 20

let y = 10;
if (true) {
    let y = 20; // Different variable
    console.log(y); // 20
}
console.log(y); // 10

const z = 30;
// z = 40; // Error: Assignment to constant variable
console.log(z); // 30

Q50. Describe the Concept of Flexbox and its advantage over traditional CSS layouts.

Answer: Flexbox (Flexible Box Layout) is a CSS layout mode designed to provide an efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown. The advantages of Flexbox over traditional CSS layouts include:

  • Flexibility: Flex items can grow and shrink to fit the container.
  • Alignment: Items can be easily aligned vertically and horizontally.
  • Ordering: Items can be reordered without changing the HTML structure.
  • Responsiveness: Flexbox makes it easier to create responsive layouts.

Example:


.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1; /* Grow and shrink equally */
}
